# 巴西 3-0 海地赛后复盘

## 赛果

巴西 3-0 海地。Matheus Cunha 梅开二度，Vinicius Junior 进球并参与串联，巴西拿到本届首胜；海地两连败后已被淘汰。

这场预测的胜负方向和零封判断是对的，但对巴西上半场转化率偏保守。我们赛前担心 Neymar 缺席、体系磨合和深盘穿透，实际巴西确实不算流畅，但 Cunha 的门前效率和 Vinicius 的决定性把比分推到了 3-0。

## 比赛判断

巴西最大的积极信号不是“踢得像冠军热门”，而是关键球员把该打进的球打进了。Guardian 赛况也提到，巴西赢得安心但并不完全流畅，慢速控球和身后挑传仍是主旋律；这说明模型不能因为 3-0 就把巴西的进攻体系直接上调到满格。

更重要的变量是 Raphinha 伤退。AP/WSLS 报道里 Ancelotti 表示 Neymar 预计可以在对苏格兰时复出，但 Raphinha 的伤情要继续盯。也就是说，巴西的前场不是简单“人员越打越完整”，而是 Neymar 回归可能对冲 Raphinha 风险。

海地方面，低位抵抗没有完全崩，但进攻穿透依旧不足。下半场有更积极的换人和几次推进，可 0-3 后比赛已经进入巴西节奏管理区。

## 小组影响

巴西升至 4 分，并凭净胜球压过摩洛哥。末轮巴西 vs 苏格兰仍然关键，尤其涉及小组第一和潜在淘汰赛路径。海地 0 分且净胜球明显落后，已经基本/实际出局。
