# D/E/F组第三轮购彩建议红队审查

## Verdict

- `verdict`: `hold_for_betting_discussion_only`
- `publish_gate`: `discussion_only`

当前不满足“可执行购彩建议”门槛。核心原因不是主方向全错，而是第三轮轮换、同开球动机变化、T-75 首发缺失，以及多场普通 `SPF` 未开售/市场校准不完整，都会把热门低赔与深让盘放大成错误自信。

## 禁止进入主仓

1. `德国胜`、`德国让球相关`  
   原因：德国已基本锁定出线，第三轮最常见误差是“胜率和净胜球被高估”，不是“实力突然消失”。当前更应把概率让给平局，而不是把德国低赔当稳胆。

2. `日本 -1`、`日本胜 + 半全场/比分绑定`  
   原因：日本有优势，但瑞典必须争胜，定位球与末段冲吊尾部真实存在；`-1` 盘本身就不支持“轻松穿盘”。

3. `荷兰深让/净胜3+`、`荷兰半全场/比分串关`  
   原因：突尼斯 vs 荷兰普通 `SPF` 未开售，只有 `+2` 让球可弱校准；荷兰若领先还可能降速保核心，深盘最容易被误读成“强队必穿”。

4. `科特迪瓦净胜2+`、`科特迪瓦大胜比分`  
   原因：库拉索 vs 科特迪瓦缺同源普通 `SPF`，且科特迪瓦存在“平局通常够用”的节奏管理脚本，不宜把客强直接外推成大胜主线。

5. `美国胜` 被包装成低风险主仓  
   原因：土耳其 vs 美国模型只是美国轻优势，不是单边压制；轮换、Pulisic 负荷和第三轮节奏变化会压缩热门安全垫。

6. `巴拉圭胜` 或 `澳大利亚胜` 的单边主仓表达  
   原因：D组这场更像平局/低比分博弈，赛果分布接近，任何单边重仓都在放大小样本波动。

## 只允许娱乐仓

1. 所有 `比分`、`半全场`、`3串1及以上` 组合。
2. 所有“热门大胜”叙事：
   - 德国 `0-2/0-3`
   - 荷兰 `0-3/0-4`
   - 科特迪瓦 `0-3`
   - 日本 `2-0/3-1`
3. 所有第三轮深让玩法：
   - `日本 -1 让胜`
   - `荷兰相关净胜3+`
   - `科特迪瓦净胜2+`
   - `美国赢2球以上`
4. D组两场的任何半全场玩法。  
   原因：两场都更像低事件、末段受另一场影响放大的比赛，高方差不应伪装成稳健玩法。

## 主仓可保留但必须降注

说明：当前没有“无条件可保留”的主仓项；以下仅限主线程坚持保留时的降级口径，且仍需通过临场门禁。

1. `日本胜方向`  
   只可保留为“非穿盘、非比分、非半全场”的弱主方向；仓位至少下调 30%-50%，并明确保留平局/瑞典尾部。

2. `荷兰胜方向`  
   仅在普通 `SPF` 恢复开售且首发无大轮换时，才允许从 `hold` 降为“主仓可保留但降注”；否则继续停留在 `discussion_only`。

3. `科特迪瓦胜方向`  
   仅在普通 `SPF` 补齐、首发稳定且未出现大幅降速信号时，允许小幅保留；不追让球、不追大胜。

4. `德国不败/德国胜讨论方向`  
   如果主线程一定要保留德国方向，只能降为轻仓讨论，不得再写“稳胆”或“低赔打底”。

## 临场撤单 / 换单条件

1. `T-75` 官方首发仍缺失。  
   直接维持 `hold_for_betting_discussion_only`，不升级执行。

2. 强队出现 `2名及以上核心轮换/保护`。  
   适用对象：德国、荷兰、日本、美国、科特迪瓦。出现则撤掉热门深盘和比分单，热门胜方向也要继续降注。

3. 普通 `SPF` 仍未开售、页面停售、或同源口径冲突。  
   特别是 `突尼斯 vs 荷兰`、`库拉索 vs 科特迪瓦`，若还只有让球盘，则不能把它们放进主仓执行。

4. 热门方向赔率临场明显漂移但无官方伤停解释。  
   若热门胜面隐含概率被动下修约 `5%` 以上，先视为信息泄漏风险，撤单或降级为纯讨论。

5. 让球盘口向“不支持穿盘”的方向重定价。  
   例如热门让胜回报突然抬高、让负压低，说明市场更担心窄胜或不胜，应撤掉深盘/比分单。

6. 临场确认比赛目标转为“保平/保净胜球/控节奏”。  
   一旦主线程或数据线程确认该脚本，应优先撤掉大胜、半全场和高比分玩法。

## 让球盘口误读风险

1. `日本 -1` 不是“日本大胜支持票”，更接近“日本优势存在，但穿盘并不稳”。
2. `突尼斯 +2 / 荷兰相关让球` 只能约束荷兰大胜尾部，不能替代普通 `SPF` 市场。
3. `库拉索 +2` 的存在说明市场并不把科特迪瓦 3球以上当成低风险主线。
4. `土耳其 +1 / 美国相关让球` 不支持把美国写成轻松两球胜。
5. `巴拉圭 -1` 对应受让侧更稳，说明这场更像低比分与平局管理，不是巴拉圭可重仓穿盘。

## 比分 / 半全场高方差提醒

1. 第三轮同组同时开球，会把 70 分钟后的策略切换放大成高方差尾部。
2. 已出线/平局够用球队最容易在领先后降速，直接破坏大胜比分与“胜/胜”半全场。
3. 必须赢的一方会在末段拉高开放度，但这会同时抬高对手反击进球，不等于单边大胜。
4. 泊松主线对红牌、早球、追分和门将高波动情景仍偏平滑，所以比分单与半全场只能放娱乐仓。

## 红队结论给主线程的可转述版本

- 这批 D/E/F 第三轮方案当前只能当 `discussion_only`，不能包装成可执行主仓单。
- 该拦截的不是“热门一定不赢”，而是“热门低赔、深让、大胜比分”被说得太稳。
- 若临场没有 `T-75` 首发、普通 `SPF`、最终伤停和一致赔率口径，就继续 `hold`，不升级到执行建议。
