## 分层口径

- 保守容错：不放 3 串及以上进主仓，靠受让覆盖和少量二串一把净回报推到约 150。
- 均衡组合：主仓仍以二串一为主，娱乐仓允许少量让平/精确净胜球。
- 进取组合：满足 300+ 回报目标，但高断点明显；只适合讨论回报弹性，不适合当命中率优先方案。

## 红队校验

- 让球平、比分、半全场只按高方差娱乐项处理。
- 德国 1.36、美国 1.66 不能被包装成低风险双锚。
- 临场如果德国、荷兰、美国、日本任一核心轮换超过预期，需要重算组合。
