# D/E/F组第三轮8套扩展返奖方案红队审查

## Verdict

- `verdict`: `revise`
- `publish_gate`: `discussion_only`
- `execution_gate`: `hold_for_betting_discussion_only`

红队核心判断：把 100 元模板从“讨论型 A/B/C 三套”扩成“保守2、均衡3、进取3”并把理论净回报目标抬到 `150/200/300+`，最大的风险不是方向错误，而是为了追返奖被动增加：

1. 串关断点数量  
2. 深让盘与“净胜刚好一球/两球”的误读  
3. 第三轮轮换与同开球动机变化  
4. 比分/半全场这类高方差玩法占比  
5. 未开售场次被混入主仓

## 允许保留

仅允许保留“保护型、非大胜叙事、非高精度比分”的底层方向，且只能继续作为讨论口径：

1. `058 日本(-1) 让负 / 瑞典+1覆盖`
2. `059 巴拉圭(-1) 让负 / 澳大利亚+1覆盖`
3. `055 德国胜` 仅能作为小额辅助项，不能当主仓锚点
4. `056 库拉索(+2)方向` 只能作为平衡风险的候选，不得因为高理论返奖而升级成主轴

## 需降级到娱乐仓

1. 所有 `3串1`、`4串4`、`4串11`  
   说明：这些不是“均衡提回报”，而是实质性增加断点。最多只能放娱乐仓。

2. 所有 `比分`、`半全场`  
   说明：第三轮同开球导致 70 分钟后脚本漂移大，比分和半全场高方差不应拿来补主仓回报。

3. 所有“刚好净胜一球/两球”的精确让球路径  
   包括：
   - `057 突尼斯(+2) 让平 = 荷兰刚好赢2球`
   - `060 土耳其(+1) 让平 = 美国刚好赢1球`
   说明：这类玩法本质上是精确比分近亲，不应进主仓。

4. 所有“热门大胜”延伸玩法  
   包括德国净胜2+、荷兰净胜3+、科特迪瓦净胜2+、日本穿 `-1`。

## 需删除 / 替换

1. 删除任何把 `056` 或 `057` 普通 `SPF` 当作可执行主仓的方案  
   原因：当前未开售或未补齐，不能假设可买。

2. 删除任何保守型方案中的 `3串1`、`4串4`、`4串11`  
   原因：保守方案的返奖目标若靠多串实现，定义已经自相矛盾。

3. 删除任何把 `055 德国1.36` 与 `060 美国1.66` 同时当作“双低赔安全锚”的主仓设计  
   原因：两队都有第三轮降速/轮换风险，低赔叠加不等于低波动。

4. 删除任何把 `057 让平`、`060 让平` 放进主仓的设计  
   原因：这不是稳健玩法，而是精确净胜球数投注。

5. 替换所有“为了冲 300+ 把比分/半全场塞进均衡方案”的写法  
   替换原则：要么明确标注为娱乐仓，要么直接删除，不得混入主仓。

## 提高返奖目标的红队边界

1. `保守2`  
   不得靠 3 串或以上实现目标；若理论净回报要冲 `150`，也只能来自 2 串以内的小幅抬升，不能引入比分、半全场、精确净胜球数。

2. `均衡3`  
   可以出现 2 串为主、少量 3 串为辅，但 3 串只能放娱乐仓，不得写成主仓核心。

3. `进取3`  
   可以保留高波动串关，但必须明确是“娱乐仓主导”的高方差模板，不能再沿用“主仓70、娱乐30”去包装成稳健结构。

## 临场撤单 / 换单条件

1. `T-75` 官方首发缺失：整套方案继续停留 `discussion_only`。
2. 德国、荷兰、日本、美国出现 `2名及以上核心轮换/保护`：撤掉对应热门胜、深让、比分和半全场。
3. `056/057` 仍无普通 `SPF`：不得把相关常规胜负写进执行模板。
4. 临场赔率对热门方向明显降温且无公开伤停解释：先视为信息不对称，撤单或降级娱乐仓。
5. 同组另一场实时路径若确认“平局足够/小负可接受/已无追净胜球动力”：立即撤掉大胜、穿盘和半全场。

## 必须修正项

1. 不要把 `150/200/300+` 写成“目标回报”，只能写成“理论上限”，并同步写出断点数量与高方差来源。
2. 8 套方案必须把 `主仓` 和 `娱乐仓` 的玩法边界写清楚，不能用高方差票抬主仓回报。
3. 未开售场次、缺普通 `SPF` 场次、缺首发确认场次，全部只能 `discussion_only`，不能伪装成可执行票。
4. 所有“刚好净胜一球/两球”的让球平玩法，不得进入主仓。
5. 若主线程继续保留进取方案，必须明确其本质是高波动娱乐模板，而不是收益更优的理性升级版。
