# Tactical Framework

## Team Shape

Track:

- nominal formation
- in-possession structure
- out-of-possession structure
- common alternate shapes
- asymmetries, such as one fullback inverting while the other holds width
- role dependencies caused by specific players

## Attacking Phases

Build-up:

- goalkeeper involvement
- center-back split width
- pivot behavior
- fullback height
- preferred progression zones
- direct-play outlet

Chance creation:

- central overloads
- wide crossing volume and quality
- cutback patterns
- third-man runs
- striker link play
- shot profile

Transition attack:

- first forward pass target
- carry threat
- runner count
- counterattack speed
- vulnerability if the first outlet is unavailable

## Defensive Phases

Pressing:

- press height
- pressing triggers
- man-oriented vs zonal behavior
- weak-side risk
- goalkeeper press behavior

Settled block:

- line height
- compactness
- protection of zone 14
- wide defending
- aerial defending
- late-box defending

Defensive transition:

- rest-defense shape
- counterpress aggression
- tactical foul tendency
- recovery pace

## Set Pieces

Attacking:

- primary takers
- delivery type
- aerial targets
- blocker/screen patterns
- second-ball structure

Defending:

- zonal/man mix
- goalkeeper command
- near-post protection
- second-ball vulnerability
- foul risk

## Coach Profile

Track:

- preferred system and flexibility
- selection stability
- substitution timing and purpose
- game-state behavior when leading/drawing/trailing
- tournament pragmatism
- willingness to rotate
- historical knockout-game risk posture

## Confidence Cues

High confidence:

- repeated across competitive matches
- supported by current squad availability
- visible under the same coach

Medium confidence:

- repeated but from mixed contexts
- affected by one or two uncertain players

Low confidence:

- based on friendlies only
- old coach or old squad
- contradicted by recent injuries or role changes
