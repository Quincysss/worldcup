## 一句话定位

哥伦比亚是 K 组出线热门，Luis Diaz 的边路爆点、James 的创造和南美对抗强度都很突出，但破低位效率和情绪控制要谨慎看。

## 球队档位

K 组上中游。哥伦比亚纸面不如葡萄牙豪华，但它的边路、定位球、反击和中场对抗足够强，尤其对乌兹别克斯坦这种首次世界杯球队，会有明显压迫优势。

## 教练与战术身份

Nestor Lorenzo 的哥伦比亚大概率以 4-2-3-1 / 4-3-3 展开。James 负责创造和定位球，Luis Diaz 负责左路爆破，Arias、Duran 和中锋轮换提供禁区冲击。

## 核心球员角色

| 球员 | 角色 |
| --- | --- |
| Luis Diaz | 左路爆点、纵深和一对一 |
| James Rodriguez | 前场创造、定位球和节奏 |
| Jhon Arias | 右路/中路连接、跑动和压迫 |
| Jhon Duran | 中锋冲击、终结和替补爆点 |
| Jefferson Lerma | 中场对抗、二点球和防守覆盖 |
| Davinson Sanchez | 中卫速度、对抗和防空 |
| Camilo Vargas | 门线和禁区指挥 |

## 关键问题

哥伦比亚如果早进球，会把比赛带进自己舒服的转换和边路冲击；如果乌兹别克斯坦守得很低，哥伦比亚需要靠 James 的定位球和 Luis Diaz 的单点质量破局。

## 强项

- 定位球脚法

- 反击速度
