## 一句话定位

日本首轮逼平荷兰，很提气；但第二轮的关键不是“还能不能打强队”，而是能不能把突尼斯低位撬开。

## 球队档位

F 组中上竞争队。技术纪律和执行力很成熟，首轮后模型评价上调。

## 教练与战术身份

森保一的日本强调团队移动、边路换位、压迫纪律和快速转换。Kubo、Mitoma 是撕开防线的关键，Endo 和 Morita 负责中场稳定。

## 核心球员角色

| 球员 | 角色 |
| --- | --- |
| Takefusa Kubo | 右路/肋部创造和定位球 |
| Kaoru Mitoma | 左路一对一突破 |
| Daichi Kamada | 前场连接和二线插上 |
| Wataru Endo | 中场防守和节奏稳定 |
| Hidemasa Morita | 中场衔接和控球 |
| Ko Itakura | 中卫出球和组织 |
| Ayase Ueda | 禁区终结和前场压迫 |

## 首轮后修正

2-2 荷兰后，日本的强队对抗能力、心理和战术执行力上调；但对突尼斯低位时，破密集和定位球防守是另一套题。

## 强项

- 战术纪律
- 边路一对一
- 中前场技术
- 压迫执行
- 反击第一脚质量

## 短板

- 身体对抗
- 定位球防守
- 面对低位时的终结效率
- 边路核心体能
- 热度上升后的盘口风险

## 模型提示

`transition_attack=high`，但 `low_block_breaking=key_vs_tunisia`。不能直接把逼平荷兰外推成必然打开突尼斯。

## 预测前复核清单

Kubo/Mitoma 体能、Endo 状态、突尼斯阵型、官方技术报告、即时盘口。
