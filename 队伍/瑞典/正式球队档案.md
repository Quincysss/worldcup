## 一句话定位

瑞典首轮 5-1 太亮眼了，锋线配置也确实吓人；但下一场对荷兰，才是真正检验这套火力含金量的时候。

## 球队档位

F 组中上竞争队。大胜后积分、净胜球和信心都很漂亮，但需要做对手质量折扣。

## 教练与战术身份

Graham Potter 的瑞典可以打得更灵活：Isak 和 Gyökeres 提供双前锋冲击，Kulusevski 负责推进和创造，Elanga 给边路速度。

## 核心球员角色

| 球员 | 角色 |
| --- | --- |
| Alexander Isak | 纵深、脚下连接和终结 |
| Viktor Gyökeres | 力量冲击、压迫和禁区终结 |
| Dejan Kulusevski | 右路/中路推进和创造 |
| Anthony Elanga | 边路速度和反击 |
| Lucas Bergvall | 中场年轻推进变量 |
| Victor Lindelöf | 防线经验和出球 |

## 首轮后修正

5-1 突尼斯后，瑞典进攻均值和锋线权重上调；同时上调过热风险，尤其不能把弱化对手的结果无限外推。

## 强项

- 双前锋火力
- 身体和速度
- 反击纵深
- 边路冲击
- 首轮大胜气势

## 短板

- 中场控制稳定性
- 面对荷兰控球时防线承压
- 防线速度
- 首轮样本过热
- 双前锋身后保护

## 模型提示

`attack_ceiling=high`，`public_bias_risk=high_after_5_1`。对荷兰时必须给防守压力和控球劣势单独折扣。

## 预测前复核清单

Isak/Gyökeres/Kulusevski 状态、荷兰首发、官方技术报告、即时盘口、中场保护方案。
