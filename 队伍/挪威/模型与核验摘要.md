# 挪威队模型与核验摘要

更新时间：2026-06-10 15:45 +08:00

## 模型状态

当前模型状态：`partial`

原因：

- 结构化 team packet 尚未完全落地。
- 战术包尚未完全落地。
- 赔率快照需要完整 outcome set 才能稳定计算 overround 与去水概率。
- 核心球员可用性需要拆成 `available_for_squad`、`available_to_start`、`minutes_risk`。

因此现在不输出最终概率，只输出模型框架和初版方向。

## 挪威建模关键

挪威不能只按普通中游队建模，也不能只按球星队建模。正确做法是拆成两层：

1. 长期强度层：Elo、FIFA 排名、Opta/power rating、近期表现、阵容身价。
2. 核心可用性层：哈兰德、厄德高、索尔洛特、努萨、鲍勃的状态和首发情况。

哈兰德主要影响：

- 进球转化率
- 高比分尾部
- 反击兑现能力

厄德高主要影响：

- 阵地战创造质量
- 低位防守破解
- 定位球质量

两人都正常首发时，挪威的进攻上限需要上调；但如果其中一人有负荷限制，不能只小幅下调胜率，还要下调进球均值和多球胜出的概率。

## 红队 verdict

红队当前口径：

- 一般性看高挪威：`revise`
- 投注倾向：`hold`

不是因为挪威不强，而是因为最容易误判的地方太明显：

- 哈兰德和厄德高名气容易放大整队稳定性。
- 俱乐部角色不能直接搬到国家队。
- 防线、边路和转换防守没有进攻线那么硬。
- 首战赔率已经偏热，价格可能已经吃掉不少挪威优势。

## 主线程使用规则

后续我输出挪威分析时必须遵守：

- 可以说“挪威上限高”，但必须补一句“体系兑现率和防线保护仍要看比赛验证”。
- 可以倾向挪威小组出线，但不能写成“稳出线”。
- 可以倾向挪威胜伊拉克，但不能把低赔胜方向当成投注价值。
- 所有涉及投注的表达，必须等 T-75m 首发和最新盘口。
